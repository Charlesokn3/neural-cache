# Neural Cache
A cyberpunk-themed personal budget tracker with piggy bank and savings goals.

## Live app
https://charlesokn3.github.io/neural-cache

## Features
- Income and expense tracking with 50/30/20 budget split
- Piggy bank for general savings
- Savings goals that track as expenses
- Weekly, biweekly and monthly views
- Data synced via Supabase -- works across any device

## Install

**Desktop (Chrome/Edge)**
Open the link -> click the install icon in the address bar -> Install

**iPhone (Safari)**
Open the link -> tap Share -> Add to Home Screen

**Android (Chrome)**
Open the link -> tap the three dots menu -> Add to Home Screen

## Stack
HTML - CSS - JavaScript - Supabase
